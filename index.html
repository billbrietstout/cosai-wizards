CoSAI Wizards

Five vanilla-web assessment tools for practitioners working with the CoSAI Shared Responsibility Framework, CSA AICM, OWASP LLM Top 10, NIST AI RMF, and MITRE ATLAS. Apache-2.0, no build step, no backend, no trackers, every answer stays in your browser.

License: Apache-2.0 Runtime: HTML + CSS + JS Dependencies: jsPDF (self-hosted)
